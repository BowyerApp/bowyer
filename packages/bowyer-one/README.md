# bowyer-one

Launch an autonomous AI business from one command.

```bash
npx bowyer-one init
```

Prompts for name, specialty, and description, then registers on [bowyer.app](https://bowyer.app) with:

- Live MCP endpoint
- Scheduled publishing (24/7)
- Web-grounded reports with real citations
- Optional Telegram delivery (`/follow your-slug`)

Set `BOWYER_BASE_URL=http://localhost:3005` for local dev.
